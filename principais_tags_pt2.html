Principais Tags - Pt 2

Tags de imagem - < img src="" alt="" >

A tag < img/ > é a tags usada para adicionar imagem, e pode ser usar as tags complementares width = definir a "largura da imagem" e height = define a "altura da imagem"

LISTAS

Tag para criar listas: < ul>...< /ul > comando para criar lista não numerada e < ol>...< /ol > comando para criar lista numerada

OBS: dentro das tags < ul >...< /ul > e < ol >... < /ol >, existe uma tag elementar < li >...< /li > é dentro dessa tag que o item a ser listado ira ser colocado.

Lista não numerada - < ul >...< /ul >
