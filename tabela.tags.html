tags para tabelas

Tag: < table >...< /table >

Para iniciar a construção da tabela é necessario utilização da Tag < table >...< /table >; é ela que vai ser o esqueleto da tabela... é dentro de < table > que irão as outras tags para a formação da tabela.

tag: < thead >...< /thead >

a tag < thead > é utilizada para sinalizer que nela ira ser inserido o cabeçalho da tabelha. Dentro de thead ira a tag < tr>...< /tr>.

TAG: < tr >...< /tr >

A tag < tr>...< /tr> é usada para definir uma linha de uma tabela.

dentro dessa linha, você coloca celulas que podem ser cabeçalho (< th >) ou dados (< td >).

TAG's: < th >...< /th > é < td >...< /td >.

As tags < th> e < td > são colocadas entre a tag (< tr >...< /tr >).

    -> < th >...< /th >: Define uma célula de cabeçalho, geralmente exibida em negrito e centralizada.
    -----------------------------------------------------------------------------------------------------------
    -> < td >...< /td>: A tag < td >...< /td > define uma celula de dados, que contem o conteúdo/formação da tabela.

Tag: 1,< tbody >...< /tbody >

A tag < tbody >...< /tbody > é usada para agrupar o corpo de uma tabela, separando-o dos cabeçalhos (< thead >) e rodapé (< tfoot >).

EXEMPLO:

                    < table>
                        < thead>
                                < tr>
                                    < th>
                                        
                                    < /th>
                                < /tr>
                        < /thead>

                        < tbody>
                                < tr >
                                    < td>

                                    < /td>
                                < /tr >
                        < /tbody>

                    < /table>
