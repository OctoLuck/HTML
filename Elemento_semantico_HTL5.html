Elemento semantico < header >...< /header >

A Tag header é o cabeçalho da pagina, é dentro do comando dessa teg que é utilizado o comando (< nav >...< /nav >) para virar uma barra de nevegação.

Elemento semantico < main >...< /main >

É dentro da tag (< main >...< /main >) que ira ser feito o corpo da pagina, é nessa tag que ira as Tag's < h1 >...< /h1 >), < blockquote >...< /blockquote >, etc...

Elemento semantico < section >...< /section >

O elemento semantico (< section >...< /section >) é utilizado dentro da Tag main, ela é utilizado para por sessões dentro do corpo da pagina como por exemplo: por uma sessão que seja um portifolio, blogue ou uma sessão de apresentação pessoal. 

##Para uma tag ser identificada é preciso colocar um identificador unico um ID, então depois de colocar a tag < section >...< /section >, é necessario utilizar a tag de identificação que é 
        < section id="Nome/Numero Identificação">...< /section >.

Elemento semantico < aside >...< /aside >

É dentro da tag (< aside >...< /aside >) que ira ser colocado a barra lateral.

Elemento semantico < footer >...< /footer >

É dentro da tag (< footer >...< /footer >) que é colocado a barra de rodapé.

tag : < article >...< /article >

O elemento < article >...< /article > é usado para representar uma seção independente dentro de MAIN, como se fosse uma nova mini-pagina

Elemento: < figure >...< /figure >

A tag elementar < figure >é utilizada para envolver tudo que é relacionado a tal imagem na tag (< img src="" alt="" >). Então o mando é: 
                    < figure >
                    < img src="" alt="" >
                    < /figure >

** Para complementar e deixar mais organizado a aparencia da pagina na tag < figure>...< /figure >, e quiser deixar uma legenda proporcional a imagem pode se usar o elemento < figcaption >...< /figcaption > para por uma legenda na imagem. 

                exemplo:
                < figure >
                < img src="" alt="" >
            < figcaption > Aqui Legenda Imagem< /figcaption >
                < /figure >
